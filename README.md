# TimeTableGenerator

Генератор расписаний: автоматическое составление учебных/рабочих расписаний с учётом ограничений (аудитории, преподаватели, группы, временные слоты).

## Статус

Проект на стадии инициализации. Структура каталогов и документация заложены для дальнейшей разработки.

## Возможности (план)

- Импорт исходных данных (группы, дисциплины, преподаватели, аудитории)
- Настройка жёстких и мягких ограничений
- Автоматическая генерация расписания
- Экспорт результата (JSON, CSV, печатный вид)

## Структура репозитория

```
docs/          — описание предметной области и требований
src/           — исходный код
tests/         — тесты
examples/      — примеры входных данных
```
